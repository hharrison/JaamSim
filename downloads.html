Current JaamSim Version: 2014-30

JaamSim is distributed with all dependencies in a single file; just download and run.

Requirements: Java 6 or higher, graphics driver supporting OpenGL 3.0 (OpenGL 2.1 is supported with some degraded functionality)

NOTE: on OSX we only support Java 6. Further compatibility is a work in progress.
